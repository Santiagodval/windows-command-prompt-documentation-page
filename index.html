What is the windows command prompt

Command Prompt is a command line interpreter application available in most Windows operating systems. It's used to execute entered commands. Most of those commands automate tasks via scripts and batch files, perform advanced administrative functions, and troubleshoot or solve certain kinds of Windows issues.

open the command prompt

You can open the cmd by:

  • Searching it on the windows browser
  • Typing cmd on the task manager
  • Opening the run box with Windows + R and typing cmd
  • Create a shortcut in the desktop shortcut
  • Anchor it to the bottom bar

"md" & "mkdir" commands

Creates a folder (directory)

Usage

md folder-name
mkdir folder-nam

"rm" & "rmdir" commands

Removes a directory

Usage

rm folder-name
rmdir folder-name
rm /s folder-name
rmdir /s folder-name

"copy" command

copies a file from one location to another

Usage

copy filepath-from filepath-to

"move" command

Moves a file from one folder to another

Usage

move folder1\file.txt folder2\

"ren" & "rename" commands

Changes the name of a file

Usage

ren file1 file2

"del" command

Deletes one or more files

Usage

del filename

"exit" command

Exits batch script or current command control

Usage

exit

"echo" command

Used to display a message or to turn off/on messages in batch scripts

Usage

echo message

"type" command

Displays the content of a text files

Usage

type myfile.txt

"fc" command

Compares two files and displays the difference between them

Usage

fc file1 file2

"cls" command

Clears the screen

Usage

cls

"help" command

Provides more details about DOS/Command Prompt commands

Usage

help (lists all commands)
